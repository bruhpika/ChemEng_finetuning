# ChemE-LLM: 6-Agent Pipeline — Complete Technical Specification

> **Audience:** A capable Python developer with no prior multi-agent system experience.  
> **Stack:** OpenClaw (orchestration) · Gemini 1.5 Flash (text) · Gemini 1.5 Pro (video) · Free-tier only.  
> **Quick-start concept:** An *agent* in this pipeline is a Python function (or class) with a fixed role, a system prompt, and a set of tools. The *orchestrator* (OpenClaw) calls agents in the order defined, passes outputs forward, and checks blocking conditions (like Auditor returning `BLOCKED`).

---

## Shared Assumptions (referenced throughout)

```
N_docs     = 50   # Track A documents processed per run
N_videos   = 30   # Track B YouTube videos processed per run
N_chunks   = 500  # Expected total chunks after parsing both tracks
N_pairs    = 1000 # Expected Q&A pairs generated by Quizmaster
AUDIT_KB   = 0.10 # 10% of KB chunks sampled by Auditor (Checkpoint 1)
AUDIT_QA   = 0.05 # 5% of Q&A pairs sampled by Auditor (Checkpoint 2)

# Gemini free-tier hard limits (as of 2024)
FLASH_RPM  = 15   # gemini-1.5-flash requests per minute
FLASH_RPD  = 1500 # gemini-1.5-flash requests per day
PRO_RPM    = 2    # gemini-1.5-pro requests per minute
PRO_RPD    = 50   # gemini-1.5-pro requests per day
```

---

## Shared JSON Schema (every chunk must conform)

```json
{
  "chunk_id":   "uuid-v4-string",
  "source_type": "track_a | track_b",
  "software":   "DWSIM | MATLAB",
  "topic":      "string",
  "steps":      ["string"],
  "params":     { "param_name": "value_or_type" },
  "ui_paths":   ["string"],
  "errors":     ["string"],
  "fixes":      ["string"],
  "theory":     "string (optional, may be empty string)",
  "source_url": "string",
  "license":    "string"

```

# AGENT 1 — LIBRARIAN

**Role summary:** Discovers and validates source URLs. It is the gatekeeper of the knowledge base — nothing enters the pipeline that the Librarian has not approved for license and accessibility.

---

## 1. System Prompt

```text
[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]
You are the Librarian agent for the ChemE-LLM pipeline.

Your sole responsibility is source discovery and license validation.

Rules you must never break:
1. Only approve sources under these licenses: LGPL, MIT, Apache 2.0, CC-BY,
   or public domain. Reject anything else (including CC-BY-NC, CC-BY-SA).
2. Every URL you output must be reachable (HTTP 200). Do not guess or
   fabricate URLs.
3. For every source, you must output: url, title, license, software_tag,
   source_type (track_a or track_b).
4. If you cannot determine the license with high confidence from the page
   content, classify it as UNKNOWN and do NOT approve it.
5. You are not allowed to output any copyrighted text from source documents.
   Output metadata only.

You have access to: web_search, url_fetch, license_checker tools.
Think step by step. Return only structured JSON. No prose.
```

> **[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]**  
> Intended behavior: Register this system prompt as the agent persona in OpenClaw's agent config. The orchestrator should instantiate this agent once per pipeline run and keep it alive until `sources.csv` is written. Tool bindings (`web_search`, `url_fetch`, `license_checker`) must be declared in the agent's tool registry.

---

## 2. Tools Needed

| Tool name | Input parameters | Output format |
|---|---|---|
| `web_search` | `query: str`, `num_results: int = 10` | `list[dict]` with keys `title`, `url`, `snippet` |
| `url_fetch` | `url: str`, `timeout: int = 15` | `dict` with keys `status_code`, `content_type`, `text` |
| `license_checker` | `page_text: str`, `url: str` | `dict` with keys `license: str`, `confidence: float` |

## 3. Custom Tool Wrappers

```python
import requests
from duckduckgo_search import DDGS
import google.generativeai as genai
import time


def web_search(query: str, num_results: int = 10) -> list[dict]:
    """Search DuckDuckGo (no API key required) and return title/url/snippet list."""
    results = []
    with DDGS() as ddgs:
        for r in ddgs.text(query, max_results=num_results):
            results.append({
                "title": r.get("title", ""),
                "url": r.get("href", ""),
                "snippet": r.get("body", ""),
            })
    return results


def url_fetch(url: str, timeout: int = 15) -> dict:
    """Fetch a URL and return status code, content type, and raw text."""
    try:
        resp = requests.get(url, timeout=timeout, headers={"User-Agent": "ChemE-LLM/1.0"})
        return {
            "status_code": resp.status_code,
            "content_type": resp.headers.get("Content-Type", ""),
            "text": resp.text[:20000],  # cap at 20 KB to stay within token limits
        }
    except requests.RequestException as e:
        return {"status_code": 0, "content_type": "", "text": str(e)}


def license_checker(page_text: str, url: str) -> dict:
    """Ask Gemini Flash to identify the license of a page from its text content."""
    ALLOWED = {"LGPL", "MIT", "Apache 2.0", "CC-BY", "public domain"}
    model = genai.GenerativeModel("gemini-1.5-flash")

    prompt = f"""
You are a software license classifier. Read the following webpage text and identify 
the license of the content or software described.

URL: {url}
PAGE TEXT (first 3000 chars):
{page_text[:3000]}

Respond ONLY with a JSON object — no markdown, no explanation:
{{"license": "<license name or UNKNOWN>", "confidence": <0.0 to 1.0>}}

Known license names to use: MIT, Apache 2.0, LGPL, CC-BY, CC-BY-SA, CC-BY-NC, 
GPL, BSD, public domain, UNKNOWN.
"""
    time.sleep(4)  # respect 15 RPM flash limit
    response = model.generate_content(prompt)
    import json
    try:
        result = json.loads(response.text.strip())
    except json.JSONDecodeError:
        result = {"license": "UNKNOWN", "confidence": 0.0}

    result["approved"] = result["license"] in ALLOWED and result["confidence"] >= 0.7
    return result
```

> **Dependencies:** `pip install duckduckgo-search requests google-generativeai`

---

## 6. Failure Handling

| Failure condition | Action |
|---|---|
| `url_fetch` returns `status_code != 200` | Mark `approved=false`, log to `librarian_errors.log`, skip source |
| `license_checker` returns `confidence < 0.7` | Classify as `UNKNOWN`, mark `approved=false` |
| Gemini returns non-JSON in `license_checker` | Catch `JSONDecodeError`, return `{"license": "UNKNOWN", "confidence": 0.0, "approved": false}` |
| DuckDuckGo rate limit / timeout | Exponential backoff: wait 10s, 20s, 40s then skip query; log to `librarian_errors.log` |
| Zero approved sources found | Raise `LibrarianError("No approved sources found for {SOFTWARE}")` — orchestrator halts pipeline |

---

## 7. Gemini Quota Notes

```
Calls = N_docs (license checks for Track A URLs) + N_videos (for Track B URLs)
      = 50 + 30 = 80 Flash calls per run

At 15 RPM: 80 calls ÷ 15 = ~6 minutes of API time.
At 1500 RPD: well within daily limit.

THROTTLE RULE: Insert time.sleep(4) after every Flash call.
BATCH RULE: Do not parallelize license_checker calls — run sequentially.
```

# AGENT 2 — TRACK A PARSER

**Role summary:** Converts PDF and HTML documents (from `track_a/sources.csv`) into schema-compliant JSON chunks using Gemini Flash extraction. It is the only agent that reads raw documents.

---

## 1. System Prompt

```text
[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]
You are the Track A Parser for ChemE-LLM.

You receive chunks of text extracted from PDF or HTML documentation for 
DWSIM or MATLAB. Your job is to convert each chunk into a structured JSON 
object that strictly matches the ChemE-LLM chunk schema.

Rules:
1. Extract ONLY information that is explicitly present in the input text.
   Do not hallucinate steps, parameters, or UI paths.
2. If a field has no data in the text, use an empty list [] for list fields
   or "" for string fields. Do NOT invent placeholder content.
3. The `steps` field must contain numbered, imperative-voice instructions
   (e.g., "Open the Flowsheet Editor", "Set inlet temperature to 25°C").
4. The `ui_paths` field must use the format: "Menu > Submenu > Option".
5. Always assign a unique chunk_id using uuid4.
6. Flag any chunk where steps=[] OR ui_paths=[] in a separate field:
   "flag": "INCOMPLETE".
7. Return ONLY valid JSON. No markdown, no explanation.
```

> **[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]**  
> Intended behavior: This agent is invoked in parallel with Track B Scout by the orchestrator after Librarian completes. It reads each approved Track A source URL, extracts text, chunks it, and sends each chunk to Gemini Flash for structuring.

---

## 2. Tools Needed

| Tool name | Input parameters | Output format |
|---|---|---|
| `pdf_extractor` | `url: str` | `list[str]` (one string per page) |
| `html_extractor` | `url: str` | `str` (cleaned text) |
| `text_chunker` | `text: str`, `chunk_size: int = 1500` | `list[str]` |
| `gemini_flash_parse` | `chunk_text: str`, `software: str`, `source_url: str`, `license: str` | `dict` (chunk JSON) |
| `schema_validator` | `chunk: dict` | `bool`, `list[str]` (error messages) |

---

## 3. Custom Tool Wrappers

```python
import pdfplumber
import requests
from bs4 import BeautifulSoup
import google.generativeai as genai
import uuid, json, time, io


def pdf_extractor(url: str) -> list[str]:
    """Download a PDF from url and return a list of page text strings."""
    resp = requests.get(url, timeout=20)
    resp.raise_for_status()
    pages = []
    with pdfplumber.open(io.BytesIO(resp.content)) as pdf:
        for page in pdf.pages:
            text = page.extract_text() or ""
            pages.append(text.strip())
    return pages


def html_extractor(url: str) -> str:
    """Fetch an HTML page and return cleaned body text (no scripts/styles)."""
    resp = requests.get(url, timeout=15, headers={"User-Agent": "ChemE-LLM/1.0"})
    resp.raise_for_status()
    soup = BeautifulSoup(resp.text, "html.parser")
    for tag in soup(["script", "style", "nav", "footer", "header"]):
        tag.decompose()
    return soup.get_text(separator="\n", strip=True)


def text_chunker(text: str, chunk_size: int = 1500) -> list[str]:
    """Split text into overlapping chunks of ~chunk_size characters."""
    OVERLAP = 200
    chunks = []
    start = 0
    while start < len(text):
        end = start + chunk_size
        chunks.append(text[start:end])
        start = end - OVERLAP
    return chunks


def gemini_flash_parse(
    chunk_text: str,
    software: str,
    source_url: str,
    license_str: str
) -> dict:
    """Send one text chunk to Gemini Flash and return a schema-compliant dict."""
    model = genai.GenerativeModel("gemini-1.5-flash")
    prompt = f"""
Convert the following documentation chunk into a JSON object for the ChemE-LLM 
knowledge base. Software: {software}.

CHUNK TEXT:
{chunk_text}

Return ONLY this JSON structure (no markdown, no explanation):
{{
  "chunk_id": "{uuid.uuid4()}",
  "source_type": "track_a",
  "software": "{software}",
  "topic": "<short topic label>",
  "steps": ["<step 1>", "<step 2>"],
  "params": {{"<param_name>": "<value_or_type>"}},
  "ui_paths": ["<Menu > Sub > Option>"],
  "errors": ["<error message if any>"],
  "fixes": ["<fix description if any>"],
  "theory": "<background theory, or empty string>",
  "source_url": "{source_url}",
  "license": "{license_str}"
}}

If a list field has no data, return [].
If params has no data, return {{}}.
Add "flag": "INCOMPLETE" if steps=[] OR ui_paths=[].
"""
    time.sleep(4)  # 15 RPM Flash throttle
    response = model.generate_content(prompt)
    try:
        chunk = json.loads(response.text.strip())
    except json.JSONDecodeError:
        # Return a skeleton flagged for manual review
        chunk = {
            "chunk_id": str(uuid.uuid4()),
            "source_type": "track_a",
            "software": software,
            "topic": "PARSE_ERROR",
            "steps": [], "params": {}, "ui_paths": [],
            "errors": ["gemini returned non-json"],
            "fixes": [], "theory": "",
            "source_url": source_url,
            "license": license_str,
            "flag": "PARSE_ERROR"
        }
    return chunk


REQUIRED_KEYS = {
    "chunk_id", "source_type", "software", "topic",
    "steps", "params", "ui_paths", "errors", "fixes",
    "theory", "source_url", "license"
}

def schema_validator(chunk: dict) -> tuple[bool, list[str]]:
    """Validate a chunk dict against the required ChemE-LLM schema keys and types."""
    errors = []
    missing = REQUIRED_KEYS - set(chunk.keys())
    if missing:
        errors.append(f"Missing keys: {missing}")
    if not isinstance(chunk.get("steps", None), list):
        errors.append("steps must be a list")
    if not isinstance(chunk.get("ui_paths", None), list):
        errors.append("ui_paths must be a list")
    if not isinstance(chunk.get("params", None), dict):
        errors.append("params must be a dict")
    return (len(errors) == 0, errors)
```

## 7. Gemini Quota Notes

```
Assumptions:
  N_docs = 50 documents
  avg_pages_per_doc = 20 pages
  avg_chunk_size = 1500 chars → ~375 tokens
  chunks_per_page = 1.5 (overlap)
  total_chunks = 50 × 20 × 1.5 = 1500 Flash calls

WARNING: 1500 calls/day is right at the Flash free-tier daily limit (1500 RPD).
This means Track A Parser MUST be run alone on a given day, not combined with
other heavy Flash usage.

MITIGATION STRATEGIES:
1. REDUCE N_docs: Start with 20 documents per run (600 calls), leave headroom.
2. DEDUPLICATE before parsing: If a URL was already parsed in a previous run
   (check chunks_dwsim.json for source_url), skip it.
3. CACHE: Save raw extracted text to disk before calling Gemini, so a re-run
   on the same doc doesn't re-fetch.
4. THROTTLE: time.sleep(4) between every Flash call (enforces <15 RPM).
5. BATCH runs across multiple days if N_docs > 25.
```

# AGENT 3 — TRACK B SCOUT

**Role summary:** Searches YouTube for walkthrough videos, filters them by quality criteria, then calls Gemini 1.5 Pro with each video URL to extract structured knowledge chunks. This is the only agent that uses `gemini-1.5-pro` — treat Pro quota as precious.

---

## 1. System Prompt

```text
[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]
You are the Track B Scout for ChemE-LLM.

Your job is to find YouTube tutorial videos about DWSIM or MATLAB simulations 
and extract structured knowledge from them using Gemini's video understanding.

Rules:
1. Only accept videos that meet ALL of these criteria:
   - Language: English only
   - Duration: ≤ 30 minutes
   - View count: ≥ 500
   - Content type: walkthrough OR troubleshooting (not concept-only lectures)
2. For each accepted video, call gemini_pro_video_extract with the YouTube URL.
3. Return schema-compliant JSON chunks. source_type must be "track_b".
4. Do not fabricate steps or UI paths not visible in the video.
5. Return ONLY valid JSON arrays. No prose.
```

> **[OPENCLAW-SPECIFIC — verify in OpenClaw docs before use]**  
> Intended behavior: Run in parallel with Track A Parser after Librarian completes. The orchestrator should enforce a hard cap of `PRO_RPD=50` total video calls per day across both software targets combined.

---

## 2. Tools Needed

| Tool name | Input parameters | Output format |
|---|---|---|
| `youtube_search` | `query: str`, `max_results: int = 20` | `list[dict]` with keys `video_id`, `title`, `duration_sec`, `views`, `url` |
| `video_filter` | `videos: list[dict]`, `max_duration_sec: int`, `min_views: int` | `list[dict]` (filtered) |
| `gemini_pro_video_extract` | `video_url: str`, `software: str`, `license_str: str` | `list[dict]` (chunk objects) |

---

## 3. Custom Tool Wrappers

```python
from yt_dlp import YoutubeDL
import google.generativeai as genai
import uuid, json, time


def youtube_search(query: str, max_results: int = 20) -> list[dict]:
    """Search YouTube using yt-dlp (no API key required) and return video metadata."""
    ydl_opts = {
        "quiet": True,
        "extract_flat": True,
        "default_search": f"ytsearch{max_results}",
        "skip_download": True,
    }
    results = []
    with YoutubeDL(ydl_opts) as ydl:
        info = ydl.extract_info(query, download=False)
        for entry in info.get("entries", []):
            results.append({
                "video_id": entry.get("id", ""),
                "title": entry.get("title", ""),
                "duration_sec": entry.get("duration", 0) or 0,
                "views": entry.get("view_count", 0) or 0,
                "url": f"https://www.youtube.com/watch?v={entry.get('id', '')}",
            })
    return results


def video_filter(
    videos: list[dict],
    max_duration_sec: int = 1800,
    min_views: int = 500
) -> list[dict]:
    """Filter videos by duration and view count thresholds."""
    return [
        v for v in videos
        if v["duration_sec"] <= max_duration_sec
        and v["views"] >= min_views
        and v["duration_sec"] > 0
    ]


def gemini_pro_video_extract(
    video_url: str,
    software: str,
    license_str: str = "YouTube Standard License"
) -> list[dict]:
    """
    Call Gemini 1.5 Pro with a YouTube video URL using the video_metadata content part.
    This is NOT a standard chat call — it uses Gemini's native video understanding.
    Returns a list of schema-compliant chunk dicts extracted from the video.
    """
    model = genai.GenerativeModel("gemini-1.5-pro")

    # Gemini 1.5 Pro accepts YouTube URLs directly as video_metadata parts
    # via the google.generativeai SDK using a Part object.
    # Reference: https://ai.google.dev/gemini-api/docs/vision#youtube
    video_part = {
        "video_metadata": {"video_url": video_url}
    }

    prompt_text = f"""
Watch this {software} tutorial video and extract structured knowledge chunks.

For each distinct procedure, workflow, or troubleshooting sequence you observe,
produce ONE JSON chunk. Return a JSON array of chunks.

Each chunk must have this exact structure:
{{
  "chunk_id": "<generate a uuid>",
  "source_type": "track_b",
  "software": "{software}",
  "topic": "<topic label>",
  "steps": ["<step 1 as seen in video>", "<step 2>"],
  "params": {{"<param>": "<value observed>"}},
  "ui_paths": ["<Menu > Sub > Option as seen in video>"],
  "errors": ["<error shown in video, if any>"],
  "fixes": ["<fix demonstrated in video, if any>"],
  "theory": "<any theory explained by narrator, or empty string>",
  "source_url": "{video_url}",
  "license": "{license_str}"
}}

Rules:
- Only include what is ACTUALLY shown or spoken in the video.
- steps must be concrete, imperative actions ("Click...", "Enter...", "Select...").
- If steps or ui_paths are not observable, return [] for those fields.
- Return ONLY the JSON array. No prose, no markdown fences.
"""
    # CRITICAL: gemini-1.5-pro is rate limited to 2 RPM on free tier
    time.sleep(32)  # 60s / 2 RPM = 30s minimum; use 32s for safety margin

    response = model.generate_content([
        {"text": prompt_text},
        video_part
    ])

    try:
        # Strip any accidental markdown fences
        raw = response.text.strip().lstrip("```json").rstrip("```").strip()
        chunks = json.loads(raw)
        if not isinstance(chunks, list):
            chunks = [chunks]
    except json.JSONDecodeError:
        chunks = [{
            "chunk_id": str(uuid.uuid4()),
            "source_type": "track_b",
            "software": software,
            "topic": "PARSE_ERROR",
            "steps": [], "params": {}, "ui_paths": [],
            "errors": ["gemini-pro returned non-json"],
            "fixes": [], "theory": "",
            "source_url": video_url,
            "license": license_str,
            "flag": "PARSE_ERROR"
        }]

    # Assign fresh chunk_ids and validate source_url
    for c in chunks:
        c["chunk_id"] = str(uuid.uuid4())
        c["source_url"] = video_url
    return chunks
```

> **Dependencies:** `pip install yt-dlp google-generativeai`

You receive JSON chunk files from Track A and Track B and produce unified,

Rules:
1. Deduplication key = SHA-256 hash of (topic.lower().strip() + software.lower()).
2. When two chunks share the same dedup key:
   - Track A wins for fields: params, ui_paths
   - Track B wins for fields: steps, fixes
   - TIE-BREAKING: If the winning track's field is [] or null,
     use the losing track's value instead.
3. All other fields (errors, theory, chunk_id) come from the higher-confidence
   track (prefer Track A for theory, prefer Track B for errors).
4. Log every conflict to conflicts.log with the chunk_id, field, and which
   track won.
5. Output two files: master_kb_dwsim.json and master_kb_matlab.json.
6. This agent makes ZERO Gemini API calls.

```python
import json, hashlib, os, glob
from pathlib import Path


def load_chunks(directory: str, software: str) -> list[dict]:
    """Load all chunk JSON files for a given software from a directory."""
    pattern = os.path.join(directory, f"chunks_{software.lower()}.json")
    chunks = []
    for filepath in glob.glob(pattern):
        with open(filepath, "r", encoding="utf-8") as f:
            data = json.load(f)
            if isinstance(data, list):
                chunks.extend(data)
    return chunks


def _dedup_key(chunk: dict) -> str:
    """Compute SHA-256 deduplication key from topic and software."""
    raw = chunk.get("topic", "").lower().strip() + chunk.get("software", "").lower()
    return hashlib.sha256(raw.encode()).hexdigest()


def _tiebreak(winning_val, losing_val):
    """Return losing_val if winning_val is empty/null; else return winning_val."""
    if winning_val is None or winning_val == [] or winning_val == {}:
        return losing_val
    return winning_val


def dedup_and_merge(
    track_a_chunks: list[dict],
    track_b_chunks: list[dict]
) -> tuple[list[dict], list[str]]:
    """
    Merge Track A and Track B chunks with field-level conflict resolution.
    Returns (merged_chunks, conflict_log_lines).
    """
    conflict_log = []
    # Index Track A chunks by dedup key
    a_index: dict[str, dict] = {}
    for c in track_a_chunks:
        key = _dedup_key(c)
        a_index[key] = c

    # Index Track B chunks by dedup key
    b_index: dict[str, dict] = {}
    for c in track_b_chunks:
        key = _dedup_key(c)
        b_index[key] = c

    all_keys = set(a_index.keys()) | set(b_index.keys())
    merged = []

    for key in all_keys:
        a = a_index.get(key)
        b = b_index.get(key)

        if a and not b:
            merged.append(a)
            continue
        if b and not a:
            merged.append(b)
            continue

        # Both tracks have this chunk — apply conflict resolution
        base = a.copy()  # start with Track A as base

        # Track A wins: params, ui_paths
        for field in ("params", "ui_paths"):
            resolved = _tiebreak(a.get(field), b.get(field))
            if resolved != a.get(field):
                conflict_log.append(
                    f"[TIEBREAK] chunk_id={a['chunk_id']} field={field} "
                    f"track_a was empty → used track_b value"
                )
            base[field] = resolved

        # Track B wins: steps, fixes
        for field in ("steps", "fixes"):
            resolved = _tiebreak(b.get(field), a.get(field))
            if resolved != b.get(field):
                conflict_log.append(
                    f"[TIEBREAK] chunk_id={b['chunk_id']} field={field} "
                    f"track_b was empty → used track_a value"
                )
            else:
                conflict_log.append(
                    f"[CONFLICT] chunk_id={a['chunk_id']} field={field} "
                    f"winner=track_b"
                )
            base[field] = resolved

        # errors: prefer Track B (more likely to show real runtime errors)
        base["errors"] = b.get("errors") or a.get("errors") or []

        # theory: prefer Track A (docs are more authoritative)
        base["theory"] = a.get("theory") or b.get("theory") or ""

        # Preserve chunk_id from Track A for consistency
        merged.append(base)

    return merged, conflict_log


def write_master_kb(chunks: list[dict], output_path: str) -> None:
    """Write merged chunk list to a JSON file, creating directories if needed."""
    Path(output_path).parent.mkdir(parents=True, exist_ok=True)
    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(chunks, f, indent=2, ensure_ascii=False)
```

import google.generativeai as genai
import json, time, random
from pathlib import Path


def generate_pairs_for_chunk(
    chunk: dict,
    category: str,
    n_pairs: int = 3
) -> list[dict]:
    """Call Gemini Flash to generate n_pairs fine-tuning pairs of a given category."""
    model = genai.GenerativeModel("gemini-1.5-flash")

    CATEGORY_HINTS = {
        "how-to": "procedural questions about how to perform a task step by step",
        "troubleshoot": "questions about diagnosing errors and applying fixes",
        "parameter": "questions asking for specific parameter values, units, or ranges",
        "conceptual": "questions asking for theory, background, or explanation of why",
    }

    prompt = f"""
Generate {n_pairs} fine-tuning instruction pairs from this knowledge chunk.
Category: {category} ({CATEGORY_HINTS[category]})
Software: {chunk.get('software', 'UNKNOWN')}

CHUNK:
{json.dumps(chunk, indent=2)}

Return a JSON array of {n_pairs} objects, each with:
{{"instruction": "<question>", "input": "<context>", "output": "<answer>"}}

Rules:
- instruction: natural student question
- input: brief context (e.g., "In DWSIM, setting up a distillation column")
- output: accurate, complete answer grounded ONLY in the chunk above
- Do NOT invent information not in the chunk
- Return ONLY the JSON array. No prose, no markdown fences.
"""
    time.sleep(4)  # 15 RPM Flash throttle
    response = model.generate_content(prompt)
    try:
        raw = response.text.strip().lstrip("```json").rstrip("```").strip()
        pairs = json.loads(raw)
        # Tag each pair with its category for counting
        for p in pairs:
            p["_category"] = category
        return pairs if isinstance(pairs, list) else []
    except json.JSONDecodeError:
        return []  # Failed chunk — caller handles retry


def count_by_category(pairs: list[dict]) -> dict[str, int]:
    """Count pairs per category using the _category tag."""
    counts = {"how-to": 0, "troubleshoot": 0, "parameter": 0, "conceptual": 0}
    for p in pairs:
        cat = p.get("_category", "")
        if cat in counts:
            counts[cat] += 1
    return counts


def check_balance(
    counts: dict[str, int],
    total: int,
    min_share: float = 0.20
) -> list[str]:
    """Return list of category names below min_share of (total / 4)."""
    ideal_per_cat = total / 4
    threshold = ideal_per_cat * min_share  # 20% of ideal
    return [cat for cat, n in counts.items() if n < threshold]


def write_jsonl(pairs: list[dict], output_path: str) -> None:
    """Write pairs to a JSONL file, stripping internal _category tags."""
    Path(output_path).parent.mkdir(parents=True, exist_ok=True)
    with open(output_path, "w", encoding="utf-8") as f:
        for p in pairs:
            clean = {k: v for k, v in p.items() if not k.startswith("_")}
            f.write(json.dumps(clean, ensure_ascii=False) + "\n")
```

import google.generativeai as genai
import requests, json, time, random
from pathlib import Path


def sample_items(items: list, sample_rate: float) -> list:
    """Return a random sample of items at the given rate (0.0–1.0)."""
    k = max(1, int(len(items) * sample_rate))
    return random.sample(items, min(k, len(items)))


def fetch_source_text(url: str) -> str:
    """Fetch URL content and return up to 5000 characters of text."""
    try:
        resp = requests.get(url, timeout=15, headers={"User-Agent": "ChemE-LLM/1.0"})
        resp.raise_for_status()
        return resp.text[:5000]
    except Exception as e:
        return f"FETCH_ERROR: {e}"


def gemini_verify_chunk(chunk: dict, source_text: str) -> dict:
    """Ask Gemini Flash to verify chunk fields are grounded in source_text."""
    model = genai.GenerativeModel("gemini-1.5-flash")
    prompt = f"""
You are a fact-checker for a RAG knowledge base.

SOURCE TEXT (from {chunk.get('source_url', 'unknown')}):
{source_text}

CHUNK TO VERIFY:
steps: {json.dumps(chunk.get('steps', []))}
params: {json.dumps(chunk.get('params', {}))}
ui_paths: {json.dumps(chunk.get('ui_paths', []))}

For each of the three fields (steps, params, ui_paths), check whether the 
values are supported by the source text above.

Return ONLY this JSON (no prose, no markdown):
{{
  "verified": true | false,
  "unverified_fields": ["steps" | "params" | "ui_paths"]
}}

"verified" is true only if ALL three fields are supported.
"unverified_fields" lists any field where values cannot be confirmed.
If source_text starts with "FETCH_ERROR", return verified=false, 
unverified_fields=["FETCH_ERROR"].
"""
    time.sleep(4)
    response = model.generate_content(prompt)
    try:
        return json.loads(response.text.strip())
    except json.JSONDecodeError:
        return {"verified": False, "unverified_fields": ["GEMINI_PARSE_ERROR"]}


def gemini_verify_pair(pair: dict, kb_chunks: list[dict]) -> dict:
    """Ask Gemini Flash to verify a Q&A pair is grounded in the KB chunks."""
    model = genai.GenerativeModel("gemini-1.5-flash")
    # Find the most relevant chunk by topic keyword match (simple heuristic)
    instruction = pair.get("instruction", "")
    output = pair.get("output", "")
    # Sample up to 3 chunks to keep prompt size manageable
    relevant = random.sample(kb_chunks, min(3, len(kb_chunks)))

    prompt = f"""
Verify that the following Q&A fine-tuning pair is factually grounded in the 
provided knowledge base chunks. 

Q&A PAIR:
instruction: {instruction}
output: {output}

KB CHUNKS (up to 3 relevant):
{json.dumps([{k: v for k, v in c.items() if k in ("topic","steps","params","ui_paths","fixes")} for c in relevant], indent=2)}

Is the output supported by the chunks? Return ONLY:
{{"verified": true | false, "reason": "<one sentence>"}}
"""
    time.sleep(4)
    response = model.generate_content(prompt)
    try:
        return json.loads(response.text.strip())
    except json.JSONDecodeError:
        return {"verified": False, "reason": "GEMINI_PARSE_ERROR"}


REQUIRED_CHUNK_KEYS = {
    "chunk_id","source_type","software","topic",
    "steps","params","ui_paths","errors","fixes",
    "theory","source_url","license"
}

REQUIRED_PAIR_KEYS = {"instruction", "input", "output"}


class Auditor:
    """Reusable auditor called by orchestrator at two pipeline checkpoints."""

    def __init__(self, kb_chunks: list[dict] = None):
        self.kb_chunks = kb_chunks or []

    def run(
        self,
        items: list[dict],
        mode: str,  # "kb" or "qa"
        sample_rate: float = None,
        output_path: str = "cheme-llm/data/eval/qa_report.md"
    ) -> dict:
        """
        Run audit on sampled items. Returns status dict with PASS or BLOCKED.
        mode='kb': audits knowledge base chunks (sample_rate=0.10)
        mode='qa': audits Q&A pairs (sample_rate=0.05)
        """
        if sample_rate is None:
            sample_rate = 0.10 if mode == "kb" else 0.05

        sampled = sample_items(items, sample_rate)
        results = []

        for item in sampled:
            result = {"item_id": item.get("chunk_id", item.get("instruction", "?")[:30])}

            # Schema check
            required = REQUIRED_CHUNK_KEYS if mode == "kb" else REQUIRED_PAIR_KEYS
            missing = required - set(item.keys())
            result["schema_valid"] = len(missing) == 0
            result["missing_keys"] = list(missing)

            # Grounding check
            if mode == "kb":
                source_text = fetch_source_text(item.get("source_url", ""))
                verify = gemini_verify_chunk(item, source_text)
                result["verified"] = verify["verified"]
                result["unverified_fields"] = verify.get("unverified_fields", [])
                result["flagged"] = not verify["verified"] or not result["schema_valid"]
            else:
                verify = gemini_verify_pair(item, self.kb_chunks)
                result["verified"] = verify["verified"]
                result["reason"] = verify.get("reason", "")
                result["flagged"] = not verify["verified"] or not result["schema_valid"]

            results.append(result)

        total = len(results)
        flagged = sum(1 for r in results if r["flagged"])
        error_rate = flagged / total if total > 0 else 0.0
        status = "BLOCKED" if error_rate > 0.05 else "PASS"

        report = {
            "mode": mode,
            "total_checked": total,
            "flagged": flagged,
            "error_rate": round(error_rate, 4),
            "status": status,
            "details": results
        }

        write_qa_report(report, output_path)
        return report


def write_qa_report(report: dict, output_path: str) -> None:
    """Write audit results as a Markdown report file."""
    Path(output_path).parent.mkdir(parents=True, exist_ok=True)
    lines = [
        f"# Auditor Report — Mode: {report['mode']}",
        f"",
        f"| Metric | Value |",
        f"|---|---|",
        f"| Total checked | {report['total_checked']} |",
        f"| Flagged | {report['flagged']} |",
        f"| Error rate | {report['error_rate']:.2%} |",
        f"| Status | **{report['status']}** |",
        f"",
        f"## Flagged Items",
        f"",
    ]
    for r in report["details"]:
        if r["flagged"]:
            lines.append(f"- `{r['item_id']}` — schema_valid={r['schema_valid']}, "
                         f"verified={r.get('verified', '?')}")
            if r.get("missing_keys"):
                lines.append(f"  - Missing keys: {r['missing_keys']}")
            if r.get("unverified_fields"):
                lines.append(f"  - Unverified fields: {r['unverified_fields']}")
            if r.get("reason"):
                lines.append(f"  - Reason: {r['reason']}")

    with open(output_path, "w", encoding="utf-8") as f:
        f.write("\n".join(lines))
```

## 6. Failure Handling

| Condition | Action |
|---|---|
| `fetch_source_text` fails (network error) | Return `"FETCH_ERROR: ..."` string; `gemini_verify_chunk` treats FETCH_ERROR as `verified=False` |
| Gemini returns non-JSON in either verify function | Return `{"verified": False, "unverified_fields": ["GEMINI_PARSE_ERROR"]}` |
| `total_checked == 0` (empty sample) | Set `error_rate=0.0`, status=`PASS`, log warning in report |
| Auditor called but KB/JSONL file is missing | Raise `AuditorError("Input file not found: {path}")` — orchestrator halts |
| `error_rate > 0.05` | Return `status="BLOCKED"` — orchestrator is responsible for halting the pipeline on this signal |

## 7. Gemini Quota Notes

```
Checkpoint 1 (mode="kb"):
  Sampled = 10% of N_chunks = 0.10 × 500 = 50 Flash calls

Checkpoint 2 (mode="qa"):
  Sampled = 5% of N_pairs = 0.05 × 1000 = 50 Flash calls

Total Auditor Flash calls per run = 100.

Combined with other agents:
  Librarian:      80 calls
  Track A Parser: ~500 calls (with combined-category optimization)
  Quizmaster:    ~560 calls
  Auditor:        100 calls
  ─────────────────────────
  TOTAL:        ~1240 Flash calls per full pipeline run

This is within the 1500 RPD Flash free-tier limit — but with little margin.
Recommended: Do not run the full pipeline more than once per day.
Split across multiple days if you increase N_docs or N_chunks.

THROTTLE: time.sleep(4) between every Flash call in all Auditor verify functions.
```

def run_pipeline():
    # ── Step 1: Librarian ─────────────────────────────────────────────────────
    log.info("Starting Librarian...")
    # [OPENCLAW-SPECIFIC] agent.run("librarian", inputs={...})
    # Framework-agnostic equivalent: call librarian_agent() function
    librarian_agent()  # writes sources.csv to track_a/ and track_b/
    log.info("Librarian complete.")

    # ── Step 2: Track A Parser + Track B Scout in parallel ────────────────────
    log.info("Starting Track A Parser and Track B Scout in parallel...")
    errors = []

    def run_track_a():
        try:
            track_a_parser_agent()
        except Exception as e:
            errors.append(f"Track A Parser failed: {e}")

    def run_track_b():
        try:
            track_b_scout_agent()
        except Exception as e:
            errors.append(f"Track B Scout failed: {e}")

    t_a = threading.Thread(target=run_track_a)
    t_b = threading.Thread(target=run_track_b)
    t_a.start(); t_b.start()
    t_a.join();  t_b.join()

    if errors:
        for e in errors:
            log.error(e)
        raise RuntimeError("One or more parallel agents failed — see logs.")
    log.info("Both parsers complete.")

    # ── Step 3: Merger ─────────────────────────────────────────────────────────
    log.info("Starting Merger...")
    merger_agent()
    log.info("Merger complete.")

    # ── Step 4: Auditor Checkpoint 1 (KB) ─────────────────────────────────────
    log.info("Auditor Checkpoint 1 (mode=kb)...")
    kb_dwsim = json.load(open("cheme-llm/data/master_kb/master_kb_dwsim.json"))
    kb_matlab = json.load(open("cheme-llm/data/master_kb/master_kb_matlab.json"))

    result_1 = auditor.run(
        items=all_kb, mode="kb",
        output_path="cheme-llm/data/eval/qa_report_checkpoint1.md"

    log.info(f"Checkpoint 1: {result_1['status']} (error_rate={result_1['error_rate']:.2%})")
    if result_1["status"] == "BLOCKED":
        raise PipelineBlockedError(
            f"KB audit failed — error_rate={result_1['error_rate']:.2%}. "
            f"Review cheme-llm/data/eval/qa_report_checkpoint1.md"
        )

    # ── Step 5: Quizmaster ─────────────────────────────────────────────────────
    log.info("Starting Quizmaster...")
    quizmaster_agent(kb_chunks=all_kb)
    log.info("Quizmaster complete.")

    # ── Step 6: Auditor Checkpoint 2 (Q&A) ────────────────────────────────────
    log.info("Auditor Checkpoint 2 (mode=qa)...")
    pairs = [json.loads(line) for line in open("cheme-llm/data/finetune/train.jsonl")]

            f"Review cheme-llm/data/eval/qa_report_checkpoint2.md"
        )

    log.info("✅ Pipeline complete. JSONL ready at cheme-llm/data/finetune/train.jsonl")


if __name__ == "__main__":
